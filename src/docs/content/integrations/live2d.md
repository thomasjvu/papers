# Live2D Integration

2D avatar integration using Live2D Cubism models.

## Overview

Live2D provides smooth 2D avatar animation with:

- Real-time parameter control
- Expression switching
- Motion playback
- Lightweight rendering

## Setup

See the detailed guides:

- [Live2D Setup](/docs/guides/LIVE2D_SETUP)
- [Live2D Production Ready](/docs/guides/LIVE2D_PROD_READY)

Phantasy does not ship a public branded demo model by default. Bring your own Live2D asset pack and point the Appearance workspace at your hosted or local `.model3.json`.
