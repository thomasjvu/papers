# Blockchain Plugins

Optional plugins for blockchain and Web3 functionality.

## Available Plugins

- **wallet** - Multi-chain wallet management
- **wallet-coinbase** - Coinbase integration
- **wallet-solana** - Solana blockchain support
- **agentkit** - Coinbase AgentKit integration
- **x402-gateway** - Payment gateway

::: info Installation
These are external plugins. Install via the plugins directory or npm.
:::
