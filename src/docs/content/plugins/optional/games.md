# Games Plugins

Optional plugins for games and entertainment.

## Available Plugins

- **adventure** - Text adventure game engine
- **games-chess** - Chess game integration
- **games-go** - Go game integration
- **games-omok** - Omok/Gomoku game
- **romance-affection** - Affection tracking system

::: info Installation
These are external plugins bundled as `@phantasy/plugin-games`.
:::
