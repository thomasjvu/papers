# Discord Plugin

The Discord plugin enables your agent to operate as a Discord bot.

For the admin HTTP runtime, supported features are bot auth, channels,
commands, and auto-replies. Voice chat requires the dedicated Discord.js
runtime.

## Setup

1. Create a Discord Application at [Discord Developer Portal](https://discord.com/developers/applications)
2. Enable the Bot feature
3. Copy the bot token to your environment

```env
DISCORD_BOT_TOKEN=your-token-here
```

## Configuration

See the [Discord Integration Guide](/docs/integrations/discord) for detailed setup.
